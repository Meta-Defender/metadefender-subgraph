### Init

```sh
yarn
```

### Compile contract to ABIs
```sh
yarn compile
```

### Deploy subgraphs for exmaple

```sh
yarn codegen

# local graph service
yarn create-local

yarn deploy-local

yarn remove-local

# Thegraph Hosted-Service
graph auth --product hosted-service ${AccessToken}

yarn build-example

yarn deploy-example-host

```
